# ProAuthenticator 🛡️

ProAuthenticator is a next-generation open-source 2FA authenticator for Android focused on privacy, security, and offline-first design.

Built using modern Android engineering practices including Jetpack Compose, Kotlin, SQLCipher encrypted storage, and Multi-Module Clean Architecture.

## 🚀 Features

- Offline-first authentication
- Encrypted local database using SQLCipher
- TOTP and HOTP token generation
- Secure encrypted backups
- QR code account import
- Migration support from popular authenticator apps
- Material Design 3 UI
- Dynamic color support
- WearOS companion support
- Home screen widgets

## 🔐 Security Model

- No mandatory cloud account
- Local encrypted data storage
- Secure key handling using Android security APIs
- Protected backup and restore system
- Privacy-focused architecture
